# 声明

个人学习性质的示例代码仓库，主要留作 「备用」 和 「学习」 使用。

项目类型并不限制，可能有 Vue、React、Express、TypeScript 等等给类与前端相关的项目。

项目代码全部放置于 `codes` 目录下。以下对项目进行简单说明

## 项目说明

**base-learn**

> 对基本的 JavaScript 语法练习，以及在学习 ES6+ 相关的代码语法。

**mmo-rpg**

> 练习 TypeScript 和 NodeJS 在 MMO 游戏后端服务器上的开发思路与逻辑。  

**comment-app**

> 学习 React 框架时的 「简易评论框」 项目。

**make-redux**

> 学习 ReactRedux 时，先模仿 ReactRedux 源码实现 JavaScript 版本的简易 Redux。

**mimic-react-redux**

> 学习 ReactRedux 时，仿照 ReactRedux 原理实现的 「简易ReactRedux」 项目。

**gsap-scroll-example**

> 学习 GSAP 配合 React 框架时的练手代码。

**put_prayers**

> 使用 Vue 框架实现的 「宠物天堂」 项目。

**cafe-handwriting**

> 手搓 JavaScript 的源码实现，学习 JavaScript 底层源码原理与思路。

**evajs-example**

> 学习 evajs 时用到的源代码。

**graphql-example**

> 学习 GraphQL 时用到的源代码。需要配合 mongodb 使用。
